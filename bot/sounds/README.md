# 🔊 Bot Sound Effects

Umieść tutaj pliki `.ogg` z efektami dźwiękowymi.

## Oczekiwane pliki

| Plik | Kiedy jest wysyłany |
|------|---------------------|
| `rain.ogg` | Deszczowa pogoda (deszcz, mżawka, ulewa, burza) |
| `storm.ogg` | Burza z piorunami, śnieżyca |
| `sunny.ogg` | Słoneczna pogoda |
| `subscribe.ogg` | Potwierdzenie aktywacji subskrypcji |
| `error.ogg` | *(opcjonalnie)* Błąd pobierania danych pogodowych |

## Format plików

- Format: `.ogg` (Vorbis lub Opus)
- Zalecany czas trwania: 1–3 sekundy
- Rozmiar: poniżej 1 MB

## Uwagi

Jeśli dany plik nie istnieje, bot po prostu go pominie — bez błędów.
